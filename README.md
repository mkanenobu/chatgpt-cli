# chatgpt-cli

chat-gpt is a command line interface for OpenAI

# How to use

Set API Key to `~/.config/chatgpt-cli/config.json`

```json
{
  "api_key": "..."
}
```

And run chatgpt-cli

# For Developers

Dev

```bash
npm run dev
```

Build

```bash
npm run build
```

Publish


```bash
npm publish --access public
```
